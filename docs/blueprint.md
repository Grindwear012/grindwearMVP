# **App Name**: Thrift Plug

## Core Features:

- Product Listings: Display available thrift clothing items with images, descriptions, and prices.
- Shopping Cart: Allow users to add items to a shopping cart for purchase.
- Checkout Process: Enable users to securely purchase items in their shopping cart.
- Order Management: Allow the store owner to manage orders, track inventory, and fulfill shipments.
- Product Recommendation: Suggest products to users, driven by purchase history, item characteristics, and community buying patterns, enhanced by AI analysis.
- Customer Accounts: Allow customers to create accounts, and manage their addresses and payment information

## Style Guidelines:

- Primary color: Cool grey (#A9A9A9) for a clean, modern look.
- Background color: Off-white (#F5F5F5) to provide a neutral backdrop that lets the clothing stand out.
- Accent color: Soft blue (#ADD8E6) for calls to action and subtle highlights.
- Body font: 'Inter', a sans-serif font, for body text and headlines.
- Note: currently only Google Fonts are supported.
- Simple, line-based icons for navigation and key actions.
- Clean, minimalist layout with plenty of white space to showcase the clothing items.
